# Knowledge Base Structure

## 📁 简化目录结构

### 📥 00-Inbox

临时笔记、想法快速存放，每周整理一次

### 📚 01-Notes

- Technical/     # 技术笔记
- Career/        # 职业发展
- Projects/      # 项目记录
- Learning/      # 学习材料

### 🛠️ 02-Assets

- Templates/     # 常用模板
- Cheatsheets/  # 速查表
- Resources/     # 资源文件

### 🗄️ 03-Archive

存放不常用的历史文档

## 使用建议

1. 新内容直接扔进 Inbox
2. 每周末花30分钟整理 Inbox
3. 使用标签而不是复杂目录
4. 依靠搜索功能查找内容

## 核心原则

- 简单优于完美
- 够用就行
- 宁缺毋滥
